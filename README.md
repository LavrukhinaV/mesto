# Проект: Mesto
Интерактивная страница, куда можно добавлять фотографии, удалять их и ставить лайки.

### Технологии используемые в проекте
* JavaScript (откртыие и закрытые попапа; редактирование имени и информации о себе)
* Flex
* Grid
* Медиазапросы
